Let’s evaluate each option:

A. “As a firm's output decreases, average costs for production decrease.”  
❌ This describes *diseconomies of scale* in reverse — and is implausible as a general principle. Decreasing output usually leads to higher average costs (e.g., underutilized fixed capacity), not lower. So this is incorrect.

B. “As a firm's output increases, average costs for production increase.”  
❌ This is the definition of *diseconomies of scale*, not economies.

C. “As a firm's inputs triple, its output quadruples.”  
✅ This implies increasing returns to scale (output grows faster than inputs). When inputs increase by 3× and output increases by 4×, average cost (total cost / output) tends to fall — because you’re producing proportionally more with proportionally less input per unit. Assuming constant input prices, this reflects economies of scale (at the production function level, and typically translates to falling long-run average costs).

D. “As a firm's inputs triple, its output doubles.”  
❌ This is decreasing returns to scale (output grows slower than inputs), which would generally lead to rising average costs — i.e., diseconomies of scale.

Note: While option C describes *increasing returns to scale*, which is closely related and often underlies economies of scale, it’s important to distinguish:
- Economies of scale = ↓ average cost as output expands.
- Increasing returns to scale = ↑ output more than proportionally when all inputs increase proportionally.

In standard economic theory (especially in the long run with proportional input scaling), increasing returns to scale imply economies of scale — assuming constant input prices. So C is the best choice among the options.

✅ Correct answer: **C**

**Answer: C**
